Overview

This is the default theme for all child sites. It includes a unique home page template.

Home Page

By default, the home page displays the latest blog posts.

Changing the Home Page Layout

Use another page as the home page

For example, if you would just like to use your About page as the home page...

  1. From the main menu, click Settings and then Reading.
  2. Next to "Front page displays", choose A static page.
  3. Select which page you would like to use for the "Front page".
  4. If you are not using the blog, ignore the "Posts page".

Use the home page template (similiar to the home page of dentonbible.org)

  1. From the main menu, click Pages and then Add New.
  2. Title your page "Home".
  3. In the right column, change the "Template" to "Home".
  4. Click "Publish".
  5. Read the rest of the documentation in this section for details on how to alter the elements of the home page.

Visitor Information

Within the backend, navigate to Appearance > Denton Bible Church Settings.

From here, you can customize all of the data displayed in the Visitor's Information box.

  • Choose whether to display it or not
  • Change the title text
  • Change the "Service Times" text
  • Change the "Service Times"
  • Change the link to the location (i.e. Google Maps)
  • Change the link to find out more information

Web Splashes

A "Web Splash" is used to highlight an upcoming event, major news, or anything needing a spotlight on the home page. Each splash is required to have an image.

How to add a web splash

  1. From the main menu, click Posts and then Add New.
  2. In the Title field, enter a relevant title.
  3. Add any descriptive text into the editor.
  4. To add an image (more info):
    1. Click the media button.
    2. Select an image to upload. Images must be 636px x 187px or will otherwise be automatically cropped.
    3. After the image is finished uploading, edit the title if necessary and click Save all changes.
  5. Select the "Featured" category.
  6. Optionally choose an expiration date/time from the "Post Expirator" box.
  7. Optionally turn off the ability for visitorst to comment from the "Discussion" box.
  8. Click Publish or edit the publish date/time to schedule the post to publish at a later time.

How to remove a web splash

Web splashes can be assigned an expiration date. However, in the case that a date is not assigned or the splash needs to be removed early, simply change the Post Status from "Published" to "Draft". You can also delete the splash.
